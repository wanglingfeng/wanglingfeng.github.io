linux生成ssh公钥私钥

In Git Bash, paste the text below, substituting in your GitHub email address(在Git Bash中输入，生成公钥私钥)

[-t：密钥类型，包括 RSA 和 DSA 两种密钥；-b ： 指定密钥长度，默认2048；-C：提供一个新注释]

If you are using Git Bash, turn on ssh-agent(启动ssh-agent，生成Bourne shell 风格的命令输出)

Add your SSH key to the ssh-agent(把秘钥添加到高速缓存秘钥管理器中[ssh-agent])

Copy the SSH key to your clipboard(将公钥添加到黏贴板)
