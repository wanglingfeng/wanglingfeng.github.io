screen命令使用笔记

screen使用

新建会话: screen
显示全部会话：screen -ls
通过screen列表中的pid重新登陆到分离的screen：screen -r [pid]

处于screen中，按Ctrl + a后,接

  • d: 退出当前screen
  • w: 左下角显示有哪些会话
  • c: 新建会话
  • n: 移到下一个窗口
